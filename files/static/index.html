IP Geolocation Web Service

What is it?

FREE Web Service for searching Geolocation of IP addresses, and time zones.

The database is VERY accurate and contains the following data about IP addresses and time zones:

  • Country (code and name)
  • Region (code and name)
  • City Name
  • Zip Code
  • Latitude and Longitude
  • Timezone name, DST flag, and GMT offset

Data is served in up to three different formats:

  • CSV, XML and JSON
  • JSON callbacks are supported

Requests are simple REST URLs, allowing anyone to query from any programming language, or even small scripts using tools like wget or curl.

What about the data?

This service is powered by a FREE database, available from http://ipinfodb.com.

"This product includes GeoLite data created by MaxMind, available from http://maxmind.com/"
